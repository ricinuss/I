Como executar os scripts:

  1. Clique no botão copiar do script desejado.
  2. Abra o console do seu navegador:
    • No Chrome/Edge/Brave: Ctrl + Shift + J
    • No Firefox: Ctrl + Shift + K
  3. Se for sua primeira vez, o navegador pode bloquear a colagem. Se aparecer um aviso, digite exatamente:
    allow pasting e aperte Enter.
  4. Agora, cole o código (Ctrl + V) e aperte Enter para rodar.
